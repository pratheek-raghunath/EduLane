<p><h1><img src="https://github.com/pratheek-raghunath/EduLane/blob/main/assets/images/edulanesplash.png?raw=true" height="30" width="35" alt="react logo" />EduLane - Learning Management System App</h1></p><a name="about"></a>


EduLane enables teachers to effectively share course materials and course information 
to the students thus enabling all students to access course information more accessibly. 
The students can view all the course related information in a single dashboard enabling 
them to learn more effectively. EduLane also enables teacher-student interaction 
directly on the platform. EduLane feature separate teacher and student account with a 
tailor-made user interface for each of their requirements.
